# OG Image Assets — DreamVacati

This folder contains Open Graph images for social sharing previews.

## Required Files (1200 × 630 px JPEG)

All images should be JPEG format at 1200×630px for full Facebook/LinkedIn/Twitter compatibility.

| Filename | Used by |

| `montego-bay-jamaica-og.jpg` | montego-bay-jamaica.html |

## Design Specs

- **Dimensions:** 1200 × 630 px
- **Background:** Deep navy (#03045E) or destination-themed gradient
- **Typography:** Bold destination name + DreamVacati wordmark bottom-right
- **Colors:** Use DreamVacati palette — teal (#2EC4B6), horizon blue (#0077B6), Jamaica green (#006B3F)
- **File size:** Target < 200 KB per image (compress with Squoosh or similar)

## Quick Generation Options

- Canva: Use 1200×630 template, apply DreamVacati colors
- Figma: Community OG templates available
- Squoosh (squoosh.app): For compression after design
