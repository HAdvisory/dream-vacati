# Pinterest / Social Assets — DreamVacati

This folder holds Pinterest-optimized vertical graphics and social sharing images.

## Pinterest Specs

- **Dimensions:** 1000 × 1500 px (2:3 ratio — Pinterest optimal)
- **Format:** JPEG or WebP
- **File size:** < 500 KB
- **Text overlay:** Destination name + key hook in large, legible type
- **Branding:** DreamVacati logo + dreamvacati.com URL at bottom

## Design Notes

- Use vertical composition (portrait, not landscape)
- Bold title text at top or center — readable at small size
- Color palette: Jamaica green (#006B3F), ocean blue (#0077B6), deep navy (#03045E)
- Avoid cluttered text — max 2 lines of text overlay
- High contrast between text and background
